### Landing page

Landing page for Stacking up

### Running in dev

Install gatsby-cli
```
npm install --global gatsby-cli
```

Clone project
```
git clone https://github.com/ISPP-2022/ISPP-2022.github.io.git
```

Install deps and run
```
npm i && npm run develop
```

### Deploying to ghpages

```
npm run deploy
```
